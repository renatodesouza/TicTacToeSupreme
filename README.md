# Tic-Tac-Toe Supreme 🎮

Projeto desenvolvido em Kotlin com Jetpack Compose para consolidar conceitos de arquitetura Android e gerenciamento de estado.

## ✨ Funcionalidades
- [x] Menu principal com seleção de modos.
- [x] Jogo da Velha Tradicional (3x3).
- [x] Placar acumulativo de vitórias e empates.
- [x] Animações de escala (Pop-in) nos símbolos.
- [ ] Modo Avançado (Ultimate Tic-Tac-Toe) - *Em desenvolvimento*.

## 🛠️ Tecnologias Utilizadas
- **Linguagem:** Kotlin
- **UI:** Jetpack Compose (Material 3)
- **Arquitetura:** MVVM (Model-View-ViewModel)

## 🚀 Como rodar o projeto
1. Clone este repositório.
2. Abra no Android Studio (Ladybug ou superior).
3. Execute no emulador ou dispositivo físico.
